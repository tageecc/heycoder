# HeyCoder

> Hey，coder——每天多一点灵感。不贩卖焦虑，只分享一点点：开源项目、开发者工具与技术趣闻，轻松读、慢慢攒。

---

## 这是什么

**HeyCoder** 面向开发者，整理**值得一试的开源仓库**、好用的小工具和可读的技术话题，用 Markdown 写在本仓库里，方便你检索、引用，也欢迎一起补充。

风格上接近一份**个人策展清单**：不追求大而全，更在意好读、能上手、多一点灵感。

---

## 背景

开源每天都在更新，但很少有人能天天泡 GitHub。公众号适合**每天推一点**，仓库适合**沉淀与协作**——两边内容同调：这里存原文与链接，推送提醒你来看。

---

## 愿景

- **每天分享一点，每天成长一点**——节奏轻，不卷字数。
- **多一点灵感，少一点焦虑**——偏有趣、偏实用。
- **连接同路人**——好项目值得被看见，好分享值得被记录。

---
